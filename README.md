## This is an example of a good initial repository to start coding
It has:
 - Editor config
 - testing with mocha/chai
 - Coverage Package
 - eslint

## Install first:
npm i mocha chain --save-dev
npm i nyc --save-dev
npm i --save-dev eslint

## Run:
npm run test
npm run coverage
npm run lint

## For eslint run in VS terminar:
'./node_modules/.bin/eslint --init'
